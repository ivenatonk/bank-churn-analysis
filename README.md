# 🏦 Bank Customer Churn Analysis

## 📌 Project Overview
Analysis of bank customer churn patterns using 
10,000 customer records to identify key factors 
driving customer attrition.

## 🛠️ Tools Used
- **Excel** — Data cleaning, pivot tables, 
  calculated columns
- **MySQL** — SQL queries including CTEs 
  and Window Functions
- **Tableau Public** — Interactive dashboard

## 🔍 Key Findings
1. Germany has highest churn rate at 32.44% — 
   double France and Spain
2. Senior customers (46-60) churn at 51.12% — 
   most at risk age group
3. Customers with 4 products have 100% churn rate
4. Female customers churn 1.5x more than males 
   (25.07% vs 16.46%)
5. Churned customers have 25% higher average 
   balance (91,108 vs 72,745)
6. Inactive members churn 2x more than 
   active members (26.85% vs 14.27%)

## 💡 Business Recommendations
1. Launch immediate retention program for 
   German customers
2. Create senior-specific retention strategy
3. Investigate over-selling of 3-4 products
4. Develop gender-specific retention campaigns
5. Convert inactive members to active through 
   engagement campaigns
6. Create VIP program for high balance customers
